# 🖨️ Cómo Imprimir en Formato 80mm

## ⚠️ Importante

El navegador muestra por defecto el tamaño A4, pero puedes cambiarlo manualmente en el diálogo de impresión.

## 📋 Pasos para Imprimir Correctamente

### 1. Haz clic en "🖨️ Imprimir"

Verás el diálogo de impresión del navegador.

### 2. Haz clic en "Más ajustes" (si no está visible)

Esto mostrará todas las opciones de configuración.

### 3. Cambia el tamaño de papel

**Opción A: Si tienes impresora térmica configurada**
- En "Destino": Selecciona tu impresora térmica
- El tamaño se ajustará automáticamente a 80mm

**Opción B: Para guardar como PDF**
- En "Destino": Selecciona "Guardar como PDF"
- En "Tamaño de papel": Busca "Personalizado" o "Custom"
- Ingresa: **80mm x 297mm** (o el alto que necesites)

### 4. Ajusta los márgenes

- En "Márgenes": Selecciona **"Ninguno"** o **"Mínimo"**
- Esto aprovecha todo el ancho del papel

### 5. Verifica la vista previa

- La vista previa debería mostrar el recibo en formato vertical angosto
- Si se ve muy ancho, verifica el tamaño de papel

### 6. Imprime o Guarda

- Haz clic en "Guardar" (para PDF) o "Imprimir"

## 🎯 Configuración Recomendada

```
Destino: Tu impresora térmica o "Guardar como PDF"
Páginas: Todas
Diseño: Vertical
Color: Blanco y negro
Tamaño de papel: 80mm x 297mm (Personalizado)
Márgenes: Ninguno
Escala: 100%
```

## 💡 Consejos

### Para Chrome/Edge:

1. Abre el diálogo de impresión (Ctrl+P)
2. Haz clic en "Más configuración"
3. En "Tamaño del papel" selecciona "Personalizado"
4. Ingresa: Ancho = 80mm, Alto = 297mm

### Para Firefox:

1. Abre el diálogo de impresión (Ctrl+P)
2. Haz clic en "Configuración de página"
3. En "Formato y opciones" selecciona tamaño personalizado
4. Ingresa: 80mm x 297mm

### Para Impresora Térmica Real:

Si tienes una impresora térmica configurada en Windows:
1. Selecciónala en "Destino"
2. El tamaño se ajustará automáticamente
3. No necesitas cambiar nada más

## 📱 Alternativa: Configurar Impresora en Windows

Para que siempre use 80mm:

1. Ve a **Panel de Control** → **Dispositivos e impresoras**
2. Haz clic derecho en tu impresora térmica
3. Selecciona **"Preferencias de impresión"**
4. En **"Tamaño de papel"** selecciona **"80mm"** o **"Roll Paper 80mm"**
5. Guarda los cambios

Ahora cuando imprimas desde el navegador, usará automáticamente 80mm.

## 🔧 Si No Aparece la Opción de 80mm

Algunas impresoras térmicas necesitan drivers específicos:

1. Descarga el driver de tu impresora desde el sitio del fabricante
2. Instálalo en Windows
3. Reinicia el navegador
4. Ahora debería aparecer la opción de 80mm

## 📊 Tamaños Comunes de Papel Térmico

- **58mm**: Impresoras pequeñas (tickets de estacionamiento)
- **80mm**: Estándar (supermercados, restaurantes) ← **Recomendado**
- **110mm**: Impresoras grandes (facturas)

Nuestro recibo está optimizado para **80mm**.

---

**Nota:** El navegador siempre muestra A4 en la vista previa inicial, pero al cambiar el tamaño de papel se ajustará correctamente.
